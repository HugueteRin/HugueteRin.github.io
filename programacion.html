HOLA MUNDO

Este es un ejercicio básico donde se imprime la frase "Hola Mundo" usando la instrucción System.out.println();. Es un ejercicio de iniciación muy común en la programación y suele ser el primer programa que se escribe al aprender un nuevo lenguaje.

Código fuente

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
    }
}

Explicación del código

  • public class HolaMundo: Define una clase pública llamada HolaMundo
  • public static void main(String[] args): Método principal que se ejecuta al iniciar el programa
  • System.out.println("Hola Mundo"): Imprime el texto "Hola Mundo" en la consola

Ejercicio de operadores de Java

Este ejercicio consiste en ordenar 3 números de mayor a menor. El objetivo es encontrar la mayor cantidad de combinaciones posibles para ordenar los números utilizando operadores de comparación.

import java.util.Scanner;

public class OrdenarNumeros {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("Introduce el primer número:");
        int a = sc.nextInt();
        
        System.out.println("Introduce el segundo número:");
        int b = sc.nextInt();
        
        System.out.println("Introduce el tercer número:");
        int c = sc.nextInt();
        
        // Ordenar los números
        if (a >= b && a >= c) {
            if (b >= c) {
                System.out.println("Orden: " + a + ", " + b + ", " + c);
            } else {
                System.out.println("Orden: " + a + ", " + c + ", " + b);
            }
        } else if (b >= a && b >= c) {
            if (a >= c) {
                System.out.println("Orden: " + b + ", " + a + ", " + c);
            } else {
                System.out.println("Orden: " + b + ", " + c + ", " + a);
            }
        } else {
            if (a >= b) {
                System.out.println("Orden: " + c + ", " + a + ", " + b);
            } else {
                System.out.println("Orden: " + c + ", " + b + ", " + a);
            }
        }
        
        sc.close();
    }
}

  6. Ejercicio con DO - WHILE

    Pedir números hasta que se introduzca un negativo.

    import java.util.Scanner;

public class PedirNumeros {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int numero;
        int contador = 0;
        
        do {
            System.out.println("Introduce un número (negativo para terminar):");
            numero = sc.nextInt();
            
            if (numero >= 0) {
                contador++;
            }
        } while (numero >= 0);
        
        System.out.println("Has introducido " + contador + " números positivos o cero.");
        sc.close();
    }
}
    Explicación:

    Este ejercicio utiliza un bucle do-while para pedir números al usuario hasta que introduzca uno negativo. La diferencia con el bucle while es que el do-while siempre ejecuta el bloque de código al menos una vez, ya que la condición se evalúa al final.

ARRAYS Y COLECCIONES

Los arrays y colecciones en Java nos permiten almacenar múltiples valores en una sola variable. Son estructuras de datos fundamentales para cualquier programa que necesite manejar conjuntos de información.

Tipos de estructuras de datos en Java

  • Arrays - Estructuras de tamaño fijo que almacenan elementos del mismo tipo
  • ArrayList - Implementación de lista dinámica que puede crecer o reducirse
  • HashMap - Estructura de datos que almacena pares clave-valor
  • Set - Colección que no permite elementos duplicados

Ejercicios con Arrays

Ejercicio 1: Crear y recorrer un array

Este ejercicio muestra cómo crear un array de enteros y recorrerlo para calcular su suma y promedio.

public class EjemploArray {
    public static void main(String[] args) {
        // Declaración e inicialización de un array
        int[] numeros = {5, 10, 15, 20, 25};
        
        // Variables para la suma y el promedio
        int suma = 0;
        double promedio;
        
        // Recorrer el array y calcular la suma
        for (int i = 0; i < numeros.length; i++) {
            suma += numeros[i];
            System.out.println("Elemento " + i + ": " + numeros[i]);
        }
        
        // Calcular el promedio
        promedio = (double) suma / numeros.length;
        
        // Mostrar resultados
        System.out.println("La suma de los elementos es: " + suma);
        System.out.println("El promedio de los elementos es: " + promedio);
    }
}
Explicación:

Este ejercicio muestra cómo declarar un array de enteros, recorrerlo usando un bucle for, y realizar operaciones con sus elementos. El array tiene un tamaño fijo (5 elementos) y se accede a cada elemento mediante su índice.

Ejercicio 2: Array bidimensional (matriz)

Este ejercicio muestra cómo trabajar con arrays bidimensionales para representar una matriz.

public class MatrizEjemplo {
    public static void main(String[] args) {
        // Declarar e inicializar una matriz 3x3
        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        
        // Imprimir la matriz
        System.out.println("Matriz 3x3:");
        for (int i = 0; i < matriz.length; i++) {
            for (int j = 0; j < matriz[i].length; j++) {
                System.out.print(matriz[i][j] + " ");
            }
            System.out.println(); // Nueva línea al final de cada fila
        }
        
        // Calcular la suma de la diagonal principal
        int sumaDiagonal = 0;
        for (int i = 0; i < matriz.length; i++) {
            sumaDiagonal += matriz[i][i];
        }
        
        System.out.println("La suma de la diagonal principal es: " + sumaDiagonal);
    }
}
Explicación:

Este ejercicio muestra cómo trabajar con arrays bidimensionales (matrices). Se crea una matriz 3x3, se recorre usando bucles for anidados para mostrar sus elementos, y se calcula la suma de la diagonal principal (elementos donde i=j).

Ejercicio 3: ArrayList

Este ejercicio muestra cómo usar ArrayList, una implementación de lista dinámica en Java.

import java.util.ArrayList;

public class EjemploArrayList {
    public static void main(String[] args) {
        // Crear un ArrayList de Strings
        ArrayList nombres = new ArrayList<>();
        
        // Añadir elementos
        nombres.add("Ana");
        nombres.add("Juan");

        nombres.set(2, "Lucía");

        // Eliminar un elemento

        // Tamaño del ArrayList
        System.out.println("La lista contiene " + nombres.size() + " nombres.");
    }
}
Explicación:

Este ejercicio muestra cómo usar ArrayList, una colección dinámica que puede crecer o reducirse según sea necesario. Se muestran operaciones comunes como añadir, acceder, modificar y eliminar elementos, así como recorrer la lista con un bucle for-each.

PROGRAMACIÓN ORIENTADA A OBJETOS (MÓDULO 1)

La Programación Orientada a Objetos (POO) es un paradigma de programación que utiliza "objetos" para modelar entidades del mundo real. Java es un lenguaje fundamentalmente orientado a objetos.

Conceptos fundamentales de POO

  • Clase - Plantilla para crear objetos, define atributos y métodos
  • Objeto - Instancia de una clase
  • Encapsulamiento - Ocultar los detalles internos de implementación
  • Herencia - Mecanismo para crear clases a partir de otras existentes
  • Polimorfismo - Capacidad de un objeto para tomar muchas formas

Ejercicios de POO

Ejercicio 1: Crear una clase y objetos

Este ejercicio muestra cómo crear una clase simple y objetos a partir de ella.

// Definición de la clase Persona
class Persona {
    // Atributos
    private String nombre;
    private int edad;
    
    // Constructor
    public Persona(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }
    
    // Métodos
    public void saludar() {
        System.out.println("Hola, me llamo " + nombre + " y tengo " + edad + " años.");
    }
    
    // Getters y setters
    public String getNombre() {
        return nombre;
    }
    
    public void setNombre(String nombre) {
        this.nombre = nombre;
    }
    
    public int getEdad() {
        return edad;
    }
    
    public void setEdad(int edad) {
        if (edad >= 0) {
            this.edad = edad;
        }
    }
}

// Clase principal para probar
public class EjemploClase {
    public static void main(String[] args) {
        // Crear objetos de la clase Persona
        Persona persona1 = new Persona("Carlos", 25);
        Persona persona2 = new Persona("Laura", 30);
        
        // Llamar a métodos de los objetos
        persona1.saludar();
        persona2.saludar();
        
        // Modificar atributos usando setters
        persona1.setEdad(26);
        System.out.println(persona1.getNombre() + " ahora tiene " + persona1.getEdad() + " años.");
    }
}
Explicación:

Este ejercicio muestra cómo definir una clase con atributos privados, un constructor, métodos y getters/setters para encapsular los datos. Luego se crean objetos de esa clase y se interactúa con ellos llamando a sus métodos.

Ejercicio 2: Herencia

Este ejercicio muestra cómo implementar la herencia en Java.

// Clase base
class Animal {
    protected String nombre;
    protected int edad;
    
    public Animal(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }
    
    public void comer() {
        System.out.println(nombre + " está comiendo.");
    }
    
    public void dormir() {
        System.out.println(nombre + " está durmiendo.");
    }
}

// Clase derivada
class Perro extends Animal {
    private String raza;
    
    public Perro(String nombre, int edad, String raza) {
        super(nombre, edad); // Llamada al constructor de la clase base
        this.raza = raza;
    }
    
    public void ladrar() {
        System.out.println(nombre + " está ladrando: ¡Guau, guau!");
    }
    
    // Sobrescribir un método de la clase base
    @Override
    public void comer() {
        System.out.println(nombre + " está comiendo croquetas para perros.");
    }
    
    public void mostrarInfo() {
        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad + " años");
        System.out.println("Raza: " + raza);
    }
}

// Clase principal para probar
public class EjemploHerencia {
    public static void main(String[] args) {
        // Crear un objeto de la clase derivada
        Perro miPerro = new Perro("Bobby", 3, "Labrador");
        
        // Llamar a métodos heredados
        miPerro.comer();  // Método sobrescrito
        miPerro.dormir(); // Método heredado sin cambios
        
        // Llamar a método propio de la clase derivada
        miPerro.ladrar();
        
        // Mostrar información
        miPerro.mostrarInfo();
    }
}
Explicación:

Este ejercicio muestra cómo implementar la herencia en Java. Se crea una clase base Animal y una clase derivada Perro que hereda de Animal. La clase Perro añade nuevos atributos y métodos, y también sobrescribe el método comer() de la clase base.

Ejercicio 1: Interfaces y Polimorfismo

Este ejercicio muestra cómo implementar interfaces para lograr polimorfismo en Java.

// Definición de la interfaz
interface Figura {
    double calcularArea();
    double calcularPerimetro();
    void mostrarInformacion();
}

// Implementación para un círculo
class Circulo implements Figura {
    private double radio;
    
    public Circulo(double radio) {
        this.radio = radio;
    }
    
    @Override
    public double calcularArea() {
        return Math.PI * radio * radio;
    }
    
    @Override
    public double calcularPerimetro() {
        return 2 * Math.PI * radio;
    }
    
    @Override
    public void mostrarInformacion() {
        System.out.println("Círculo de radio: " + radio);
        System.out.println("Área: " + calcularArea());
        System.out.println("Perímetro: " + calcularPerimetro());
    }
}

// Implementación para un rectángulo
class Rectangulo implements Figura {
    private double base;
    private double altura;
    
    public Rectangulo(double base, double altura) {
        this.base = base;
        this.altura = altura;
    }
    
    @Override
    public double calcularArea() {
        return base * altura;
    }
    
    @Override
    public double calcularPerimetro() {
        return 2 * (base + altura);
    }
    
    @Override
    public void mostrarInformacion() {
        System.out.println("Rectángulo de base: " + base + " y altura: " + altura);
        System.out.println("Área: " + calcularArea());
        System.out.println("Perímetro: " + calcularPerimetro());
    }
}

// Clase principal para probar
public class EjemploInterfaces {
    public static void main(String[] args) {
        // Crear un array de figuras (polimorfismo)
        Figura[] figuras = new Figura[3];
        figuras[0] = new Circulo(5);
        figuras[1] = new Rectangulo(4, 6);
        figuras[2] = new Circulo(3);
        
        // Procesar todas las figuras de forma polimórfica
        for (Figura figura : figuras) {
            figura.mostrarInformacion();
            System.out.println("------------------------");
        }
    }
}
Explicación:

Este ejercicio muestra cómo usar interfaces para implementar el polimorfismo. La interfaz Figura define un contrato que todas las clases que la implementen deben cumplir. Luego, podemos tratar diferentes tipos de figuras de manera uniforme a través de la interfaz común.

Ejercicio 2: Clases Abstractas

Este ejercicio muestra cómo usar clases abstractas para compartir código entre subclases.

// Clase abstracta
abstract class Vehiculo {
    protected String marca;
    protected String modelo;
    protected int año;
    
    public Vehiculo(String marca, String modelo, int año) {
        this.marca = marca;
        this.modelo = modelo;
        this.año = año;
    }
    
    // Método concreto (implementado)
    public void mostrarDetalles() {
        System.out.println("Vehículo: " + marca + " " + modelo + " (" + año + ")");
    }
    
    // Método abstracto (debe ser implementado por las subclases)
    public abstract void acelerar();
    
    // Método abstracto
    public abstract void frenar();
}

// Subclase que extiende la clase abstracta
class Coche extends Vehiculo {
    private int numeroPuertas;
    
    public Coche(String marca, String modelo, int año, int numeroPuertas) {
        super(marca, modelo, año);
        this.numeroPuertas = numeroPuertas;
    }
    
    @Override
    public void acelerar() {
        System.out.println("El coche " + marca + " " + modelo + " está acelerando");
    }
    
    @Override
    public void frenar() {
        System.out.println("El coche " + marca + " " + modelo + " está frenando");
    }
    
    // Método específico de la clase Coche
    public void abrirPuertas() {
        System.out.println("Abriendo las " + numeroPuertas + " puertas del coche");
    }
}

// Otra subclase
class Motocicleta extends Vehiculo {
    private boolean tieneCajon;
    
    public Motocicleta(String marca, String modelo, int año, boolean tieneCajon) {
        super(marca, modelo, año);
        this.tieneCajon = tieneCajon;
    }
    
    @Override
    public void acelerar() {
        System.out.println("La motocicleta " + marca + " " + modelo + " está acelerando rápidamente");
    }
    
    @Override
    public void frenar() {
        System.out.println("La motocicleta " + marca + " " + modelo + " está frenando con cuidado");
    }
    
    // Método específico de la clase Motocicleta
    public void hacerCaballito() {
        System.out.println("La motocicleta " + marca + " " + modelo + " está haciendo un caballito");
    }
}

// Clase principal para probar
public class EjemploClasesAbstractas {
    public static void main(String[] args) {
        // No se puede instanciar una clase abstracta
        // Vehiculo v = new Vehiculo("Error", "No instanciable", 2023); // Esto daría error
        
        Coche coche = new Coche("Toyota", "Corolla", 2022, 4);
        Motocicleta moto = new Motocicleta("Honda", "CBR", 2023, true);
        
        // Usar métodos heredados
        coche.mostrarDetalles();
        moto.mostrarDetalles();
        
        // Usar métodos implementados (abstractos en la clase padre)
        coche.acelerar();
        coche.frenar();
        
        moto.acelerar();
        moto.frenar();
        
        // Usar métodos específicos
        coche.abrirPuertas();
        moto.hacerCaballito();
    }
}
Explicación:

Este ejercicio muestra cómo usar clases abstractas para compartir código entre subclases. La clase abstracta Vehiculo define comportamientos comunes (mostrarDetalles) y declara métodos abstractos que las subclases deben implementar (acelerar y frenar). A diferencia de las interfaces, las clases abstractas pueden contener implementaciones de métodos y atributos.

Ejercicio 3: Patrones de Diseño - Singleton

Este ejercicio muestra cómo implementar el patrón de diseño Singleton en Java.

// Implementación del patrón Singleton
class ConfiguracionApp {
    // La única instancia de la clase
    private static ConfiguracionApp instancia;
    
    // Atributos de configuración
    private String nombreApp;
    private String version;
    private boolean modoDebug;
    
    // Constructor privado para evitar instanciación directa
    private ConfiguracionApp() {
        // Valores por defecto
        nombreApp = "Mi Aplicación";
        version = "1.0.0";
        modoDebug = false;
    }
    
    // Método para obtener la única instancia (thread-safe)
    public static synchronized ConfiguracionApp getInstancia() {
        if (instancia == null) {
            instancia = new ConfiguracionApp();
        }
        return instancia;
    }
    
    // Getters y setters
    public String getNombreApp() {
        return nombreApp;
    }
    
    public void setNombreApp(String nombreApp) {
        this.nombreApp = nombreApp;
    }
    
    public String getVersion() {
        return version;
    }
    
    public void setVersion(String version) {
        this.version = version;
    }
    
    public boolean isModoDebug() {
        return modoDebug;
    }
    
    public void setModoDebug(boolean modoDebug) {
        this.modoDebug = modoDebug;
    }
    
    // Método para mostrar la configuración actual
    public void mostrarConfiguracion() {
        System.out.println("Configuración de la aplicación:");
        System.out.println("Nombre: " + nombreApp);
        System.out.println("Versión: " + version);
        System.out.println("Modo debug: " + (modoDebug ? "Activado" : "Desactivado"));
    }
}

// Clase principal para probar
public class EjemploSingleton {
    public static void main(String[] args) {
        // Obtener la instancia de configuración
        ConfiguracionApp config = ConfiguracionApp.getInstancia();
        
        // Mostrar configuración por defecto
        config.mostrarConfiguracion();
        
        // Modificar la configuración
        config.setNombreApp("Super App");
        config.setVersion("2.0.1");
        config.setModoDebug(true);
        
        System.out.println("\nDespués de modificar:");
        config.mostrarConfiguracion();
        
        // Obtener otra referencia a la misma instancia
        ConfiguracionApp otraReferencia = ConfiguracionApp.getInstancia();
        
        System.out.println("\nDesde otra referencia:");
        otraReferencia.mostrarConfiguracion();
        
        // Comprobar que ambas referencias apuntan al mismo objeto
        System.out.println("\n¿Son el mismo objeto? " + (config == otraReferencia));
    }
}
Explicación:

Este ejercicio implementa el patrón de diseño Singleton, que garantiza que una clase tenga una única instancia y proporciona un punto de acceso global a ella. Es útil para recursos compartidos como configuraciones, conexiones a bases de datos o caches. El constructor privado evita la instanciación directa, y el método estático getInstancia() controla la creación y acceso a la única instancia.

Ejercicio 4: Programación Funcional con Lambda y Streams

Este ejercicio muestra cómo usar expresiones lambda y streams para procesar colecciones de datos de manera más concisa y expresiva.

import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

class Producto {
    private String nombre;
    private double precio;
    private String categoria;
    
    public Producto(String nombre, double precio, String categoria) {
        this.nombre = nombre;
        this.precio = precio;
        this.categoria = categoria;
    }
    
    public String getNombre() {
        return nombre;
    }
    
    public double getPrecio() {
        return precio;
    }
    
    public String getCategoria() {
        return categoria;
    }
    
    @Override
    public String toString() {
        return nombre + " - $" + precio + " (" + categoria + ")";
    }
}

public class EjemploLambdaStreams {
    public static void main(String[] args) {
        // Crear una lista de productos
        List productos = Arrays.asList(
            new Producto("Laptop", 1200.0, "Electrónica"),
            new Producto("Smartphone", 800.0, "Electrónica"),
            new Producto("Libro Java", 45.0, "Libros"),
            new Producto("Teclado", 25.0, "Electrónica"),
            new Producto("Novela", 15.0, "Libros"),
            new Producto("Monitor", 200.0, "Electrónica")
        );
        
        System.out.println("Lista original de productos:");
        productos.forEach(System.out::println);
        
        // Filtrar productos de electrónica
        System.out.println("\nProductos de electrónica:");
        List electronicos = productos.stream()
                .filter(p -> p.getCategoria().equals("Electrónica"))
                .collect(Collectors.toList());
        electronicos.forEach(System.out::println);
        
        // Calcular el precio total de todos los productos
        double precioTotal = productos.stream()
                .mapToDouble(Producto::getPrecio)
                .sum();
        System.out.println("\nPrecio total de todos los productos: $" + precioTotal);
        
        // Encontrar el producto más caro
        Producto masCaro = productos.stream()
                .max((p1, p2) -> Double.compare(p1.getPrecio(), p2.getPrecio()))
                .orElse(null);
        System.out.println("\nProducto más caro: " + masCaro);
        
        // Agrupar productos por categoría y contar
        System.out.println("\nNúmero de productos por categoría:");
        productos.stream()
                .collect(Collectors.groupingBy(Producto::getCategoria, Collectors.counting()))
                .forEach((categoria, cantidad) -> System.out.println(categoria + ": " + cantidad));
        
        // Aplicar descuento del 10% a productos de más de $100
        System.out.println("\nProductos con descuento del 10% (si precio > $100):");
        productos.stream()
                .filter(p -> p.getPrecio() > 100)
                .map(p -> new Producto(p.getNombre(), p.getPrecio() * 0.9, p.getCategoria()))
                .forEach(System.out::println);
    }
}
Explicación:

Este ejercicio muestra cómo usar la programación funcional en Java con expresiones lambda y streams para procesar colecciones de datos. Los streams permiten realizar operaciones de filtrado, mapeo, reducción y agrupación de manera declarativa y concisa. Las expresiones lambda proporcionan una sintaxis compacta para implementar interfaces funcionales.

JAVA AVANZADO: MANEJO DE FICHEROS

En este módulo exploraremos cómo Java maneja los ficheros, permitiéndonos leer y escribir información persistente. El manejo de ficheros es fundamental para cualquier aplicación que necesite almacenar datos de forma permanente.

Tipos de ficheros en Java

  • Ficheros de texto - Almacenan caracteres legibles por humanos
  • Ficheros binarios - Almacenan datos en formato binario
  • Objetos serializados - Permiten guardar y recuperar objetos Java completos

Ejercicios con Ficheros de Texto

Ejercicio 1: Lectura y escritura de ficheros de texto

Este ejercicio muestra cómo leer y escribir en ficheros de texto usando las clases FileReader, FileWriter, BufferedReader y BufferedWriter.

import java.io.*;

public class ManejoFicheros {
    public static void main(String[] args) {
        // Ruta del fichero
        String rutaFichero = "datos.txt";
        
        // Escribir en un fichero
        try {
            FileWriter fw = new FileWriter(rutaFichero);
            BufferedWriter bw = new BufferedWriter(fw);
            
            bw.write("Línea 1: Esto es una prueba de escritura en fichero");
            bw.newLine();
            bw.write("Línea 2: Continuamos escribiendo datos");
            bw.newLine();
            bw.write("Línea 3: Finalizamos la escritura");
            
            bw.close();
            fw.close();
            
            System.out.println("Fichero escrito correctamente");
        } catch (IOException e) {
            System.err.println("Error al escribir en el fichero: " + e.getMessage());
        }
        
        // Leer de un fichero
        try {
            FileReader fr = new FileReader(rutaFichero);
            BufferedReader br = new BufferedReader(fr);
            
            String linea;
            System.out.println("Contenido del fichero:");
            
            while ((linea = br.readLine()) != null) {
                System.out.println(linea);
            }
            
            br.close();
            fr.close();
        } catch (IOException e) {
            System.err.println("Error al leer el fichero: " + e.getMessage());
        }
    }
}
Explicación:

Este ejercicio muestra cómo escribir y leer ficheros de texto en Java. Primero, creamos un fichero y escribimos tres líneas de texto usando FileWriter y BufferedWriter. Luego, leemos el contenido del fichero línea por línea usando FileReader y BufferedReader.

JAVA AVANZADO: SERIALIZACIÓN DE OBJETOS

La serialización es el proceso de convertir un objeto en una secuencia de bytes para almacenarlo o transmitirlo. En Java, podemos serializar objetos para guardarlos en ficheros o enviarlos a través de la red.

Ejercicio: Serialización y deserialización de objetos

Este ejercicio muestra cómo serializar y deserializar objetos en Java.

import java.io.*;

// Clase que implementa Serializable
class Estudiante implements Serializable {
    // Necesario para la serialización
    private static final long serialVersionUID = 1L;
    
    private String nombre;
    private int edad;
    private double promedio;
    
    public Estudiante(String nombre, int edad, double promedio) {
        this.nombre = nombre;
        this.edad = edad;
        this.promedio = promedio;
    }
    
    @Override
    public String toString() {
        return "Estudiante [nombre=" + nombre + ", edad=" + edad + ", promedio=" + promedio + "]";
    }
}

public class SerializacionEjemplo {
    public static void main(String[] args) {
        // Crear un objeto Estudiante
        Estudiante estudiante = new Estudiante("Ana García", 20, 8.75);
        
        // Serializar el objeto (guardarlo en un fichero)
        try {
            FileOutputStream fileOut = new FileOutputStream("estudiante.ser");
            ObjectOutputStream out = new ObjectOutputStream(fileOut);
            
            out.writeObject(estudiante);
            
            out.close();
            fileOut.close();
            
            System.out.println("Objeto serializado y guardado en 'estudiante.ser'");
        } catch (IOException e) {
            System.err.println("Error al serializar: " + e.getMessage());
        }
        
        // Deserializar el objeto (recuperarlo del fichero)
        try {
            FileInputStream fileIn = new FileInputStream("estudiante.ser");
            ObjectInputStream in = new ObjectInputStream(fileIn);
            
            Estudiante estudianteRecuperado = (Estudiante) in.readObject();
            
            in.close();
            fileIn.close();
            
            System.out.println("Objeto deserializado:");
            System.out.println(estudianteRecuperado);
        } catch (IOException | ClassNotFoundException e) {
            System.err.println("Error al deserializar: " + e.getMessage());
        }
    }
}
Explicación:

Este ejercicio muestra cómo serializar un objeto Estudiante y guardarlo en un fichero, y luego cómo deserializarlo para recuperar el objeto. La clase Estudiante implementa la interfaz Serializable, lo que permite que sus instancias sean convertidas a bytes y viceversa.

JAVA AVANZADO: CONEXIÓN A BASES DE DATOS

Java Database Connectivity (JDBC) es una API que permite la ejecución de operaciones sobre bases de datos desde el lenguaje de programación Java. JDBC actúa como un puente entre una aplicación Java y una base de datos.

Componentes principales de JDBC

  • DriverManager - Gestiona los controladores de bases de datos
  • Connection - Representa una conexión con la base de datos
  • Statement - Permite ejecutar consultas SQL
  • ResultSet - Almacena los resultados de una consulta

Ejercicio: Conexión a una base de datos MySQL

Este ejercicio muestra cómo conectarse a una base de datos MySQL, realizar consultas y actualizar datos.

import java.sql.*;

public class ConexionBD {
    // Datos de conexión
    private static final String URL = "jdbc:mysql://localhost:3306/mibasedatos";
    private static final String USUARIO = "usuario";
    private static final String CONTRASEÑA = "contraseña";
    
    public static void main(String[] args) {
        Connection conexion = null;
        Statement stmt = null;
        ResultSet rs = null;
        
        try {
            // Cargar el driver de MySQL
            Class.forName("com.mysql.cj.jdbc.Driver");
            
            // Establecer la conexión
            System.out.println("Conectando a la base de datos...");
            conexion = DriverManager.getConnection(URL, USUARIO, CONTRASEÑA);
            System.out.println("Conexión establecida con éxito!");
            
            // Crear un statement
            stmt = conexion.createStatement();
            
            // Ejecutar una consulta SELECT
            String sql = "SELECT id, nombre, email FROM usuarios";
            rs = stmt.executeQuery(sql);
            
            // Procesar los resultados
            System.out.println("Usuarios registrados:");
            while (rs.next()) {
                int id = rs.getInt("id");
                String nombre = rs.getString("nombre");
                String email = rs.getString("email");
                
                System.out.println("ID: " + id + ", Nombre: " + nombre + ", Email: " + email);
            }
            
            // Insertar un nuevo usuario (usando PreparedStatement para mayor seguridad)
            String sqlInsert = "INSERT INTO usuarios (nombre, email) VALUES (?, ?)";
            PreparedStatement pstmt = conexion.prepareStatement(sqlInsert);
            pstmt.setString(1, "Nuevo Usuario");
            pstmt.setString(2, "nuevo@ejemplo.com");
            
            int filasAfectadas = pstmt.executeUpdate();
            System.out.println(filasAfectadas + " fila(s) insertada(s)");
            
            pstmt.close();
            
        } catch (SQLException e) {
            System.err.println("Error SQL: " + e.getMessage());
        } catch (ClassNotFoundException e) {
            System.err.println("Error al cargar el driver: " + e.getMessage());
        } finally {
            // Cerrar recursos
            try {
                if (rs != null) rs.close();
                if (stmt != null) stmt.close();
                if (conexion != null) conexion.close();
                System.out.println("Conexión cerrada");
            } catch (SQLException e) {
                System.err.println("Error al cerrar: " + e.getMessage());
            }
        }
    }
}
Explicación:

Este ejercicio muestra cómo conectarse a una base de datos MySQL usando JDBC. Primero cargamos el driver, establecemos la conexión, y luego realizamos operaciones como consultar datos (SELECT) e insertar nuevos registros (INSERT). También se muestra el uso de PreparedStatement para evitar inyecciones SQL.
